# LostMistRealm - 失落迷雾之境

一款基于 **Qt 6** 的跨平台迷宫探索桌面游戏。玩家需要在随机生成的 23×23 迷宫中寻找钥匙、收集草莓、放置炸弹开辟道路，最终抵达终点。游戏包含战争迷雾系统、成就系统、AI 游戏助手等丰富功能。

## 功能特性

### 核心玩法
- **随机迷宫生成** — 使用随机化 Kruskal 算法，每局生成不同的迷宫地图
- **战争迷雾系统** — 玩家视野有限，走过的路径会逐渐被迷雾重新覆盖
- **道具系统** — 地图中随机分布钥匙、草莓和炸弹，影响通关评价
- **炸弹机制** — 可在当前位置放置炸弹，2 秒后引爆 3×3 范围内的障碍物（树木/石头）
- **生命值系统** — 被自己的炸弹波及会扣血，血量归零则死亡

### 界面与交互
- **右侧控制面板** — 实时显示玩家状态（生命值、草莓、炸弹、钥匙）
- **按键操作说明** — W/A/S/D 或方向键移动，空格放置炸弹
- **道具收集日志** — 实时显示本局拾取的道具信息
- **背景音乐 & 音效** — 支持独立控制背景音乐和游戏音效的开关

### 成就系统
- **30 项成就** — 涵盖通关次数、死亡次数、草莓收集、炸弹使用、移动步数、游戏时间等多个维度
- **隐藏成就** — 部分成就为隐藏状态，需要玩家自行探索触发条件
- **数据统计** — 持久化记录死亡次数、通关次数、草莓收集量、移动步数、游戏时间等
- **进度重置** — 支持一键清除所有统计信息和成就解锁记录

### AI 游戏助手
- 集成阿里云百炼（Qwen）大模型，支持游戏相关问答
- API Key 本地持久化存储，无需每次重复输入
- 异步通信，不阻塞主线程

### 彩蛋
- 输入神秘按键序列可触发隐藏成就

## 构建与运行

### 环境要求

- C++17 编译器（MSVC / GCC / Clang）
- CMake ≥ 3.19
- Qt ≥ 6.5（需安装 Widgets、Multimedia、Sql、Network 模块）

### 构建步骤

```bash
# 1. 在项目根目录创建构建目录
mkdir build && cd build

# 2. 配置 CMake（请将 Qt6_DIR 替换为你的 Qt 安装路径）
cmake .. -DCMAKE_PREFIX_PATH=<Qt6_DIR>

# 3. 编译
cmake --build . --config Release

# 4. 运行（确保 resources 目录可被正确访问）
./LostMistRealm
```

### 部署

项目已配置 Qt 部署脚本，可通过以下命令生成安装包：

```bash
cmake --install . --prefix ./install
```

## 成就一览

游戏共有 **30 项成就**，分为普通成就和隐藏成就两类：

- **普通成就**：基于累计统计数据解锁（通关次数、死亡次数、草莓收集、炸弹使用、移动步数、游戏时间等）
- **隐藏成就**：需要满足特殊条件才能解锁（如满血通关、无炸弹通关、输入秘籍、关闭所有声音等）

可在游戏内点击 **"成就"** 按钮查看完整的成就列表和解锁状态。
